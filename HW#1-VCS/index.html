Base

  • git config --global user.name - username input
  • git config --global user.email - email input
  • git init - creating a new repository
  • git status - checking the current status of files in the repository
  • git add - add file to ready-to-save list
  • git commit - saving the current state of the file to the repository
  • git log - shows all previously saved commits
  • git checkout - allows you to switch between commits and branches
  • git branch - used to work with branches. Allows you to show the list of branches, create new ones, rename and delete
  • git push - allows you to save a local copy on the server
  • git pull - allows you to download a copy from the server
  • git merge - allows merging branches
  • git clone - allows you to create a local copy of the remote repository

.gitignore - allows you to add files and folders to the list not displayed in checkout

Advanced

  • git revert - allows you to remove commit by creating a new commit keeping history
  • git reset - allows you to remove files from staging area and remove commits
  • git rebase - allows you to transfer branches
  • git cherry-pick - allows you to copy the commit to another branch
  • git stash - allows you to save current changes without creating a new commit and then return to them (I don’t understand why you can’t just commit)
